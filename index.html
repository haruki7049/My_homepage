パソコン

パソコン:パーソナルコンピューター、略してパソコン。2022年現在、かなりのパソコンが世に出回っている。だが、あまり普通の人はパソコンに入れ込んだりはしない。らしい… 私が好きな分野としては、パソコンを動かすために必要なOS、もう少し詳しく言うとLinuxというOSがかなり好きである。Windowsはクソ

プログラミング

プログラミング:パソコン内で頑張って動いているプログラムを作る事。私のプログラミングライフの一番最初はプログラミングスクールに通い、そこでScratchから始めた。 今はそのプログラミングスクールには通ってはいないが、そこで通っていなかったら今の自分はないだろう。今となってはある程度のプログラミング言語は読む事が出来るようになった。 最近好きなプログラミング言語はLua。かなり簡単な言語であるために、感覚的に書くことが出来た。

よく書くコードはこんなコード。 


unko = {
    name = "unkoman",
    age = 24,
    greeting = function(self)
        greeting_sentence = string.format("Hi!! I'm %s.", self.name)
        print(greeting_sentence)
    end
}

なんのコードなんやろなぁ…（すっとぼけ）

ゲーム

ゲーム:ワイの小さい頃からの趣味。パソコン、もといプログラミングを始めようと思った理由はゲームを自分で作ってみたいと思ったから。 その当時、「このゲーム、もっとこうしたらいいのになぁ」なんて考えたのがきっかけ。もう少し言うとこのゲームクソだから自分で作ろう！！　って考えた事から。先人を崇めよ、当時の俺よ… 人生で最初にやったと記憶しているゲームがマリオカートWii。最近はTitanfall2が好き。EAはクソ。はっきりわかんだね（迫真）
